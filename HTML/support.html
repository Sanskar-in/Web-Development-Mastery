❓ Frequently Asked Questions

Quick answers to common questions.

Is the app available offline?▼

Yes! Once loaded, it works without internet via PWA technology. All 100 tag explanations, themes, and the code editor work offline.

How many tags are explained?▼

100 essential and modern HTML tags, each with in-depth explanations, use cases, accessibility notes, and live code examples.

Is it completely free?▼

Yes! The HTML Mastery platform is free and part of the 2026 Open Source Education initiative. Premium e-books are available on Gumroad for those who want deeper learning.

Can I use the code examples in my projects?▼

Absolutely! All code examples are free to use in personal and commercial projects. Attribution is appreciated but not required.

How do I report a bug or request a feature?▼

You can email us, open a GitHub issue, or message us on X (Twitter). We respond to all reports within 24 hours.

Which browsers are supported?▼

All modern browsers: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+. The platform uses modern CSS and JavaScript features.

Does it store my data?▼

All data (themes, progress, editor code) is stored locally on your device via localStorage. Nothing is sent to any server. See our Privacy Policy for details.

Will CSS and JavaScript mastery guides be added?▼

Yes! CSS Full Mastery is already in development, and JavaScript Full Mastery is planned for late 2026. Stay tuned!
