❓ FAQ

Frequently Asked Questions

Quick answers to the most common questions about HTML Mastery platform.

01 How do I get started with HTML Mastery?
+
Getting started is easy! Simply navigate to the HTML Mastery homepage and explore the modules. Each module is organized from beginner to advanced topics. Start with the fundamentals section to build a solid foundation, then progress through the concepts. Every concept includes interactive code examples you can run directly in the built-in sandbox.
02 Is this platform free to use?
+
Yes! HTML Mastery is completely free and open-source. All tutorials, code examples, interactive sandboxes, and learning materials are available at no cost. We believe in making quality education accessible to everyone. If you find value in this platform, consider supporting us through donations or by sharing it with others.
03 How does progress tracking work?
+
Progress tracking is built right into the platform using your browser's local storage. As you complete concepts and run code examples, your progress is automatically saved. You can view your overall completion percentage on the dashboard. Your progress persists across sessions on the same browser — no account required!
04 Can I use the code examples in my projects?
+
Absolutely! All code examples on HTML Mastery are free to use in your personal and commercial projects. The platform is open-source and designed to help you learn by doing. Feel free to copy, modify, and integrate any code snippets into your own applications. We only ask that you don't republish the entire course as your own.
05 How do I change themes?
+
HTML Mastery supports both dark and light themes. You can toggle between them using the theme button (🌙/☀️) in the top navigation bar. Your preference is automatically saved to local storage and will persist across page visits. The platform supports multiple carefully crafted theme variations across all modules.
06 How do I report bugs or suggest features?
+
We appreciate your feedback! You can report bugs or suggest features through several channels: open an issue on our GitHub repository at github.com/Sanskar-in, send an email to supportramsandesh@gmail.com, or use the quick help cards below to get started. We review all submissions and prioritize fixes based on community impact.
07 What technology does the sandbox use?
+
Our interactive code sandbox uses the latest standards to run code directly in your browser. This ensures you're learning with modern features. The environment allows you to write and run code without setting up any local development environment.
08 How can I support this project?
+
There are many ways to support HTML Mastery! You can star our GitHub repository, share it on social media, contribute code improvements via pull requests, report bugs, write about us on your blog, or make a donation to help cover hosting and development costs. Every contribution, no matter how small, helps keep this project alive and growing.
