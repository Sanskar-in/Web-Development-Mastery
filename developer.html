🤝 Contributing Guide

How to Contribute on GitHub

  1. Fork the Repository — Visit github.com/Sanskar-in and click the "Fork" button to create your own copy of the repository.
  2. Clone Your Fork — Run git clone https://github.com/YOUR-USERNAME/Web-Development-Mastery.git to get the code on your local machine.
  3. Create a Branch — Create a feature branch with git checkout -b feature/your-feature-name to keep your changes organized.
  4. Make Your Changes — Add new concepts, fix bugs, improve documentation, or enhance the UI. Follow the existing code style and conventions.
  5. Test Your Changes — Open the HTML files in your browser to verify everything looks and works correctly across devices.
  6. Commit & Push — Run git add ., then git commit -m "Add: brief description", and git push origin feature/your-feature-name.
  7. Open a Pull Request — Go to the original repository on GitHub and click "New Pull Request". Describe your changes in detail and submit.
  8. Wait for Review — I'll review your PR, suggest changes if needed, and merge it once everything is good. Welcome to the contributors list! 🎉
