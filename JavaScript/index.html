⏳ Evolution of JavaScript

1995

Creation (Mocha → LiveScript → JavaScript)

Brendan Eich created the language in just 10 days for Netscape Navigator.

1999

ES3 (Standardization)

Regular expressions, better string handling, try/catch exception handling added.

2009

ES5 & Node.js

"strict mode", JSON support, Array methods (forEach, map, filter). Node.js brings JS to the backend.

2015

ES6 (ECMAScript 2015) Revolution

let/const, arrow functions, classes, template literals, promises, modules. The biggest update in JS history.

2017

Async/Await

ES8 introduces async/await, revolutionizing asynchronous programming and escaping "callback hell".

2020

Optional Chaining & Nullish Coalescing

ES11 brings `?.` and `??`, making property access and default values much cleaner.

2023-25

Modern JS & Ecosystem

Array mutation methods (toSorted, toReversed), Records/Tuples proposals, and TypeScript dominance.
