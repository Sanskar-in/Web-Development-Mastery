💖 Love 100% Private, Local-First Tech?

HTML Full Mastery runs completely on your browser without any telemetry or cloud databases tracking your keystrokes. Keeping this educational resource free, private, and ad-free requires community love. Consider a micro-donation today!

1

Information Collection

HTML Full Mastery is an educational web application. We do not collect any personal data. No registration is required, no cookies track you, and no analytics services monitor your behavior.

Any information you provide (such as your name in the welcome screen) is stored exclusively on your device and is never transmitted to any server.

2

Local Storage Usage

We use your browser's localStorage to save:

  • Theme preferences and color customizations
  • Your display name (if provided)
  • Code editor autosave content
  • Layout preferences (editor orientation)

This data never leaves your device. You can clear it anytime via your browser settings.

3

Third-Party Services

We load the following external resources for functionality:

  • Google Fonts — For typography (Inter, Plus Jakarta Sans, Fira Code)
  • CodeMirror CDN — For the code editor engine

These services may log standard HTTP request data (IP address, browser type). We have no control over their data practices. No personal identification is included in these requests.

4

Data Security

Since all data is stored locally on your device, your information is as secure as your device itself. We recommend:

  • Keeping your browser updated to the latest version
  • Using a secure device with proper authentication
  • Clearing localStorage if using a shared computer
5

Your Rights

You have complete control over your data:

  • Access: All data is visible in your browser's Developer Tools → Application → Local Storage
  • Delete: Clear all data by clearing your browser's localStorage for this site
  • Portability: Your data is standard JSON and can be exported
  • Opt-out: Disable JavaScript to prevent localStorage usage (some features will be limited)
