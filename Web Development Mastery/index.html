📜 Evolution of HTML

1991: HTML 1.0

Berners-Lee shares "HTML Tags", the first description of HTML with just 18 tags.

1995: HTML 2.0

The first official standard. Tables and Forms become part of the web.

1999: HTML 4.01

The web matures. CSS starts to separate design from structure.

2014: HTML5

The Modern Era. Native Audio, Video, and semantic tags change everything.

Today: Living Standard

HTML is now a "Living Standard" continuously updated by WHATWG.
